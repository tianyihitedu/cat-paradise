# 🐾 喵星乐园

> 用积分喂养你的专属小猫宠物 — 班级积分互动系统

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![HTML](https://img.shields.io/badge/HTML5-single%20file-orange.svg)

## 🎯 项目简介

喵星乐园是一款轻量级的**班级积分互动系统**，适合课堂教学场景使用。学生通过获得积分来养成虚拟猫咪宠物，积分越多，猫咪等级越高、造型越酷炫！

**安装体验：** 
直接下载index.html文件到本地，用电脑，手机或平板的自带浏览器，即可打开。也可用浏览器的书签或发送到桌面，便于更快打开。

## ✨ 功能特性

### 🐱 宠物养成
- 默认宠物名为"小豆"，可自定义命名
- 7 种造型 Emoji 可选（🐱 🐈 🐈‍⬛ 😺 🐯 🦁 🐲）
- 等级升级时宠物会播放庆祝动画

### 📊 积分系统
- **快捷互动**：摸头(+5)、摸PP(-5)、抱抱(+10)、亲亲(-10)
- **自定义积分**：支持任意正负积分输入
- **积分规则**：6 条内置规则，家长模式下可自由增删改
- 内置规则示例：
  - ✅ 回答问题 +5
  - 📖 早读打卡 +3
  - 🤝 帮助同学 +5
  - ⏰ 迟到 -3
  - 😡 打架 -10
  - 💥 破坏公物 -15

### 🏆 等级系统（7 级成长）
| 等级 | 名称 | 所需积分 | 宠物造型 |
|------|------|----------|----------|
| Lv.1 | 奶猫 | 0 | 🐱 |
| Lv.2 | 幼猫 | 100 | 🐱 |
| Lv.3 | 少年猫 | 300 | 🐈 |
| Lv.4 | 成年猫 | 600 | 🐈‍⬛ |
| Lv.5 | 精英猫 | 1000 | 😺 |
| Lv.6 | 猫王 | 1500 | 🐯 |
| Lv.7 | 喵神 | 2500 | 🦁 |

> 等级名称、积分阈值均可自定义编辑

### 🔐 双模式权限
- **学生模式**：查看积分、执行规则打卡、查看统计
- **家长模式**：密码保护进入，拥有全部管理权限
  - 编辑/添加/删除积分规则
  - 编辑等级设置
  - 修改宠物名称和造型
  - 数据重置
  - 默认密码000000


### 📋 操作日志 & 统计
- 完整的操作日志记录（时间、操作、积分变化）
- 每日/每月/年度积分统计
- 日历视图展示每日积分变化

### 💾 数据持久化
- 所有数据存储在浏览器 `localStorage`，无需服务器
- 刷新页面数据不丢失

## 🚀 使用方式

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/tianyihitedu/cat-paradise.git

# 用浏览器打开
# 直接双击 index.html 或用本地服务器：
cd cat-paradise
npx serve .
# 访问 http://localhost:3000
```

## 🛠️ 技术实现

- **纯前端单文件**：所有 HTML / CSS / JavaScript 集成在一个 `index.html` 中
- **零依赖**：无需安装任何框架或库
- **响应式设计**：适配手机和桌面端
- **Google Fonts**：使用 ZCOOL KuaiLe + Noto Sans SC 字体
- **CSS 动画**：浮动粒子背景、升级庆祝动画、按钮交互效果
